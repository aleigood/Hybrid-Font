# 说明

思源黑体与DejaVu Sans Mono的混合字体，在Sublime Text下可以完美显示中文和等宽英文，适合编程使用。

# SublimeText的建议配置

```
{
	"font_face": "Source DejaVu",
	"font_size": 11,
	"font_options": ["bold"],
}
```
